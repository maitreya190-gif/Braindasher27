# BrainDasher Spectrum 2027 - PCCOE

An immersive brain-teasing gaming experience featuring five challenging puzzle games with a gaming vibe aesthetic.

## 🎮 Games Overview

| # | Game | Difficulty | Description |
|---|------|-----------|-------------|
| 1 | Eight Queens | Medium | Classic chess logic puzzle - place 8 queens without attacks |
| 2 | Memory Master | Easy | Test your memory with dynamic pattern recognition |
| 3 | Tower of Hanoi | Medium | Move discs using recursive problem-solving |
| 4 | Knight's Tour | Hard | Navigate a chess knight across the board |
| 5 | Mastermind | Very Hard | Decipher secret codes using logic |

## 🚀 Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm start`
3. Build for production: `npm run build`

## 🎨 Design Philosophy

- **Gaming Vibe**: Dark theme with neon accents
- **Performance**: Smooth animations & quick load times
- **Accessibility**: Intuitive controls & clear visuals
- **Difficulty Progression**: Easy → Very Hard
